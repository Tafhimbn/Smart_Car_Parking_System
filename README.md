# Smart Car Parking System


## List of components:

Arduino nano

TCRT5000

IC7805

0.33uf and 0.1uf capacitor

16x2 lcd display

10 k variavle resistor

220 ohms,100ohms,10k Ohms resistor

Ferric cloride

Double sided copper clad PCB

Header pin

connecting wire

9g Servo motor

9v battery
